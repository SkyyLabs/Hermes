# Idea

LocalMacAgent is a macOS assistant that favors local execution, explicit user
intent, voice-first interaction, and chat-native workflows. It should grow in
phases so each implemented capability has tests, docs, traceability, and a clear
safety boundary.

Phase 0 creates the engineering foundation. It does not provide assistant chat,
voice capture, command automation, RAG retrieval, screen analysis, or external
integrations yet.
