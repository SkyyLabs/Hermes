# Feature 00: Foundation

## Implemented

- Python package metadata and `src` layout
- YAML plus `.env` settings precedence
- Runtime path creation and containment helpers
- Safety action classification and enforcement helpers
- Deterministic JSONL logging helper functions
- Minimal development CLI commands
- Documentation, schema outlines, memory seed files, configs, and tests

## Not implemented

Chat, voice, command automation, RAG retrieval, memory routing, background work,
screen context, external app integration, gestures, cloud APIs, and the Swift
native shell remain future roadmap phases.
